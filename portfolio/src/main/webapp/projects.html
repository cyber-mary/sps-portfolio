Projects & Experience

Tic Tac Toe

In my Programming & Methodology II class, I created a tic tac toe game throughout the semester using Java. First, I created a tic tac toe game in which two players would type the row & column of the space they would like to take. Then, an updated board would print and the game would continue until there was a winner or tie. Then, I created an AI to play against a human player. The human player would type which row and column of the space they would like to take. If the human player only had one more space left to win, then the AI would take that space, or if the AI only had one space left to win, then it would take that space. Otherwise, it would choose any random available space Lastly, I adjusted the game using Java GUI, so the player could see the tic tac toe board easily
